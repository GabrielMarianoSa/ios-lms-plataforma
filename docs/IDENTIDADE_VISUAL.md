# Identidade visual (aproximação)

Este projeto usa um tema roxo inspirado na identidade do IOS (https://ios.org.br/),
mas sem copiar o layout/HTML/CSS do site.

- Cor primária: roxo
- Tom secundário: roxo escuro
- Destaques: lilás/roxo claro em fundos sutis

Ajustes ficam centralizados em `assets/css/site.css`.
